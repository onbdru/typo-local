# Локальный Типограф (Monorepo)

Репозиторий проекта локального быстрого офлайн-типографа для русского и английского языков.

## Структура проекта

* **`core/`** — общая библиотека форматирования текста (JavaScript, без внешних зависимостей).
* **`vscode/`** — расширение для редактора VS Code.

## Документация расширений

* Подробное описание и настройки расширения VS Code находятся в файле [vscode/README.md](vscode/README.md).
